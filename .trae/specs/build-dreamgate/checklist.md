# Checklist — DreamGate

> 验收检查点。每项需在实现后逐条核验，通过后打勾。

## 报名与赛道
- [ ] 报名创意提案帖已发到 TRAE 社区报名专区，带 `生活娱乐` 标签
- [ ] 报名审核通过（领取 99 元速通月卡 + 决赛门票）
- [ ] Demo 帖赛道标签 = 报名帖赛道标签 = `生活娱乐`

## TRAE 合规（硬性要求）
- [ ] 全程代码经 TRAE IDE + web-dev skill 编写
- [ ] `trae-sessions.md` 已记录每 Task 的 Session ID（≥3 个有效）
- [ ] `docs/screenshots/` 已存 ≥3 张关键开发截图（镜门/首张AI藏品/3D走廊/分享卡/部署成功）
- [ ] Demo 帖文案含 TRAE 实践过程叙述（任务拆解/能力使用/关键 Prompt/踩坑）
- [ ] Demo 帖附 ≥3 Session ID + ≥3 截图

## 镜之门开场
- [ ] 首次打开显示居中镜之门 + 环境雾效 + 微光粒子 + "踏入梦境"提示
- [ ] 点击/滚动触发碎镜动画（碎片飞散 + 雾气扩散）并转场到主界面
- [ ] WebGL 不可用/低端设备时降级为 CSS 碎片转场且流畅

## 梦境录入
- [ ] 60 秒内可完成文字录入并保存
- [ ] 语音按钮可转写填入文本框（不支持浏览器隐藏按钮）
- [ ] 元数据（情绪/颜色/标签）未填也可保存，不阻塞
- [ ] 录入后原文立即写入 IndexedDB，AI 失败原文不丢
- [ ] 情绪选择器映射附录 B 情绪词库（~24 细粒度词）

## AI 梦境藏品生成
- [ ] 首次使用引导选择美学预设（附录 D 4 个），后续所有梦境图遵循该风格（画廊视觉一致）
- [ ] 梦境图通过 Pollinations flux + 固定 prompt 模板 + 美学预设 + seed 生成
- [ ] CSS 色调滤镜统一画廊色调，dreamlike 氛围优先于写实
- [ ] 大模型解析经 `/api/llm` 边缘函数代理（SiliconFlow），Key 不暴露前端
- [ ] 返回细粒度情绪词（映射附录 B）+ 强度 + 基调，情绪基调权重高于符号
- [ ] 符号解读映射附录 C 符号库，以概率/可能性呈现，附"需结合自身语境"说明，非断言
- [ ] 图像生成失败从种子库按「预设×情绪」匹配占位 + 重试按钮
- [ ] 大模型解析失败有规则关键词降级（仍映射附录 B/C）
- [ ] 生成过程异步渐进式呈现（不阻塞界面）

## 3D 走廊画廊（全量 3D + 双端 + 降级开关）
- [ ] 桌面端滚动驱动 Three.js 相机沿走廊漂移，滚动可前后控制
- [ ] 门前高亮 + 显示梦境标题/情绪标签
- [ ] 点击门/深入可进入梦境房间沉浸视图（大图+解析+粒子）
- [ ] 房间氛围灯光随梦境情绪变化（噩梦暗红 / 美梦暖光等）
- [ ] 雾效着色器按梦境情绪调色（可被降级开关关闭改 CSS 渐变雾）
- [ ] 移动端降级 2.5D 视差画廊（CSS 3D + GSAP + 粒子），30fps+
- [ ] 降级开关可一键将桌面端降为 2.5D（第12天自检用）
- [ ] Lenis 平滑滚动不破坏 CSS sticky
- [ ] GSAP scrub 动画绑定滚动位置可前后
- [ ] 无梦境时有空状态引导录入

## 梦境卡片分享
- [ ] 3D 可定制卡片编辑器（边框/字体/背景氛围）实时预览
- [ ] 降级开关可切 3 套预设静态模板卡片（第12天自检用）
- [ ] 可导出图片
- [ ] 生成带参链接，打开可只读查看该梦境藏品

## 情绪追踪与 streak
- [ ] streak 计数正确，断签不惩罚且显示鼓励文案
- [ ] 日历热力图按情绪着色正确
- [ ] 情绪分布可视化展示

## 隐私本地优先
- [ ] 首次 AI 生成弹窗征求同意，拒绝后纯本地可用（仅原文，无 AI 图/解析）
- [ ] 一键导出全部梦境数据为 JSON
- [ ] 全站明显位置有"娱乐与自省用途，非医疗诊断"免责声明

## 演示种子数据
- [ ] 首次进入预置 5 个精选示例梦境（含已生成藏品）
- [ ] 5 个示例含跨梦模式（如 3 个梦都出现"水"+焦虑），触发潜意识报告演示
- [ ] 评委打开即见丰满画廊，可完整体验流程
- [ ] 提供「清空示例梦境」按钮，清空后进入空状态

## 离线保底
- [ ] AI 全失败（Pollinations + SiliconFlow 都不可达）时梦境原文仍保存
- [ ] 离线时图用种子库按情绪匹配占位
- [ ] 离线时解析用规则关键词提取情绪（映射附录 B）
- [ ] UI 标注"离线模式"

## 降级开关（第 12 天自检）
- [ ] 第 12 天（7.11）执行自检，记录决策到 `docs/degradation-decisions.md`
- [ ] 移动端 2.5D 不达标 → 仅保录入+列表浏览
- [ ] 3D 分享卡不达标 → 静态模板卡片
- [ ] 雾效着色器不达标 → CSS 渐变雾
- [ ] Demo 帖如实说明降级取舍（评审重过程而非完美）

## 潜意识报告（P1）
- [ ] 累积 ≥3 条且 ≥7 天可生成周报
- [ ] 周报含记录数/主题聚类/重复符号/情绪强度变化/streak
- [ ] 跨梦模式识别高亮"过去 N 天梦到 X M 次常伴随 Y 情绪"
- [ ] 演示种子数据（跨梦模式）可触发报告演示
- [ ] 报告末尾为反思提示（非诊断）+ 免责

## 共享梦池（P1）
- [ ] 匿名投递仅上传图+情绪+符号标签，原文不上传
- [ ] 无点赞、无排行、无算法推荐
- [ ] 相似度关联可点击跳转到相似梦境

## 创作素材库（P1）
- [ ] 梦境藏品可加入灵感板并加备注
- [ ] 灵感板支持按情绪/符号/色调/相似图检索

## 部署与体验（国内托管优先）
- [ ] 腾讯云 EdgeOne Pages 静态 + 边缘函数部署可在线访问（主链接）
- [ ] Vercel 兜底备选部署可在线访问（备链接）
- [ ] 国内访问连通性验证通过（主链接优先）
- [ ] 桌面端 3D 30fps+，移动端 2.5D 30fps+（Lighthouse 审计）
- [ ] 双端核心流程（镜门→录入→生成→画廊→分享）均可走通
- [ ] 初赛 Demo 帖含演示视频/GIF + 可体验链接 + `生活娱乐` 标签 + 报名帖链接

## 数据模型与规范
- [ ] IndexedDB schema 按附录 A 实现（dreams/meta/inspirations）
- [ ] 情绪词库（附录 B ~24 词）落地为常量，录入与 AI 解析均映射
- [ ] 符号库（附录 C ~20 符号）落地为常量，AI 解析映射
- [ ] 美学预设规范（附录 D 4 预设）落地：prompt 关键词 + CSS 滤镜 + 情绪适配

## 壁垒自检（防"提示词就能替代"）
- [ ] 产品具备通用 AI 不具备的能力：累积式梦境图谱 + 统一美学一致性 + 沉浸 3D 浏览 + 共享梦池，至少 3 项落地
- [ ] 演示种子数据能体现"累积+跨梦模式识别"壁垒（非单次解析可替代）
