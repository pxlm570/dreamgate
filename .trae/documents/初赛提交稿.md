## 一、Demo 简介

**梦阈 DreamGate** 是一个本地优先的「个人潜意识档案馆」Web 应用——把转瞬即逝的梦境，变成可以反复走进、反复端详的 3D 油画画廊。

**面向谁**：关注自我探索、有记录梦境习惯的人。梦境是潜意识寄来的信，但传统日记本只能承载文字——梦阈让它变成一幅幅可以走进去的画。

**主要功能**：

| 功能 | 说明 |
|---|---|
| 🚪 镜之门 | 3D 大理石拱门开场，滚轮分幕推进，碎镜瞬间穿过门洞走进画廊 |
| 🖼️ 3D 走廊画廊 | 你的每场梦生成一幅风格统一油画，挂在 Belle Époque 走廊墙上，逐画驻足观展 |
| 🌙 梦境记录 | 文字/语音录入 → AI 生成油画 + 情绪/符号解析（暗房显影仪式） |
| 📊 跨梦报告 | 纵向发现你重复出现的意象与情绪模式（水×5、门×3…）——通用 AI 替代不了的壁垒 |
| 💌 共享梦池 | 匿名投递梦境漂流瓶，相似度共鸣连接陌生人的潜意识 |
| 🎴 梦境分享卡 | 3D 编辑器生成可分享的梦境卡片，只读链接社交传播 |

**体验地址**：

- 🌐 备链接（Vercel）：`[部署后填入 xxx.vercel.app]`

## 二、Demo 创作思路

### 灵感来源

梦是免费的 3D 电影，但醒来后往往只剩模糊的几帧。传统的梦境日记有两个硬伤：
1. **只有文字**——梦是画面的，文字记不下那种氛围
2. **只有单篇**——记了 30 场梦，却看不出自己反复在梦什么

我想要一个工具：能把梦变成画，还能纵向发现「我一直在梦什么」。

### 解决的问题

| 痛点 | 梦阈的解法 |
|---|---|
| 梦记下来只有干瘪文字 | AI 生成风格统一油画，每场梦都是一幅可走进去的画 |
| 记了一堆看不出规律 | 跨梦模式识别——重复意象、相伴情绪、连续记梦热力图 |
| 梦境是隐私，不想上传 | 本地优先架构，全部 IndexedDB，不上传服务器 |
| 平铺列表没有沉浸感 | 3D 走廊画廊，真连续穿门，画即世界 |

### 为什么做这个方向

1. **核心壁垒**：跨梦模式识别是通用 AI 替代不了的——它需要你的纵向历史数据，需要本地积累，不是一句 prompt 能复现的
2. **隐私优先**：梦境是最私密的内容之一，本地优先架构是天然护城河
3. **沉浸感**：3D 画廊 + 真连续穿门，把「翻日记」变成「走进美术馆」

## 三、技术实现

### 技术栈

```
Vite 6 · React 18 · TypeScript 5.8
three.js + @react-three/fiber（3D 渲染）
GSAP + ScrollTrigger（滚动动效）
Zustand 5（状态管理）
idb 8（IndexedDB 本地持久化）
framer-motion（UI 过渡）
```

### 核心技术亮点

#### 1. 真连续穿门（最大技术亮点）

镜之门与 3D 走廊共享**同一个持久 Canvas**（WorldPage 是 layout route，路由切换不重挂）。碎裂白闪瞬间，门内换成真实走廊——走廊显形、雾切走廊值、镜湖环境隐去，相机加速推过门洞，CameraRig 接力。**无遮罩、无静帧，穿过门就走进了美术馆**。

> 这解决了 WebGL 上下文销毁/重建导致的转场卡顿——没有两个 Canvas 并存，没有 Context Lost 风险。

#### 2. 三级兜底（保交付）

```
图像：gpt-image(/api/img) → Pollinations URL → 种子图(SVG)
解析：Edge AI → SiliconFlow → 规则解析(ruleParser)
3D：全量 3D 走廊 → 2.5D 视差 → CSS 拱门降级
```

每一级降级自动触发，评委环境即使无 Key、AI 额度耗尽，也能完整体验。

#### 3. 跨梦模式识别（核心壁垒）

不只是单场梦的解析，而是纵向聚合：
- **重复意象**：自动识别你多次梦到的符号（水、门、飞翔…）
- **相伴情绪**：哪些情绪总是一起出现
- **连续记梦**：streak 热力图 + 维度分布

这需要本地历史数据积累，是通用 AI 一句话 prompt 替代不了的。

#### 4. 移动端 2.5D 降级

3D 走廊在移动端自动降级为 2.5D 视差画廊（GSAP scrub translateZ/rotateX），保留空间纵深感。WebGL 不可用时进一步降级为 CSS 拱门。FpsSampler 运行时监测帧率，持续低帧自动降级（已优化为多窗口采样，避免滑动误触发）。

## 六、使用说明

### 快速开始

1. 打开体验链接（EdgeOne 主链接或 Vercel 备链接）
2. 首次进入会看到引导弹窗——同意后自动加载 9 条精选示例梦境
3. 滚轮推进镜之门 → 碎镜穿门 → 走进 3D 走廊画廊
4. 点击任意画作或简介牌 → 进入梦境详情
5. 点击「记录新梦」→ 输入梦境文字 → AI 生成油画 + 解析
6. 访问「情绪报告」→ 查看跨梦模式识别
7. 访问「共享梦池」→ 投递梦境漂流瓶

### 核心交互

| 操作 | 效果 |
|---|---|
| 滚轮/滑动 | 镜之门分幕推进 / 画廊逐画驻足 |
| 点击画作 | 入画 → 梦境详情页 |
| 点击「记录新梦」 | 进入录入页 → AI 生成 |
| 画廊右上角 3D/2.5D 切换 | 手动切换视图模式 |
| 语音按钮（Chrome/Edge） | 语音转文字录入 |

### 数据说明

- 全部数据存在浏览器 IndexedDB，清除浏览器数据 = 清空档案馆
- 无需登录注册，无后端数据库
- 分享链接为只读，不含你的私密数据（仅分享你选择的那场梦的摘要）
