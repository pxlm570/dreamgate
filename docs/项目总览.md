## 1. 项目是什么

**梦阈 · DreamGate** — 本地优先的「个人潜意识档案馆」Web App（TRAE AI 创造力大赛参赛作品，初赛截止 2026-07-15，已完成报名）。

核心流程：**记录梦境 → AI 生成风格统一的梦境画 + 情绪/符号解析 → 3D 走廊画廊漫游 → 分享卡导出 → 情绪连击与跨梦模式报告**。全部数据存浏览器 IndexedDB，无后端数据库。

## 2. 技术栈与命令

Vite 6 · React 18 · TypeScript 5.8 · three.js / R3F + drei · GSAP + Framer Motion + Lenis · Tailwind 3 · Zustand 5 · idb

```bash
npm run dev        # 开发服务器
npm run check      # 类型检查（改动后必跑）
npm run lint       # eslint（存量 11 个 any 类错误，零新增为过关线）
npm test           # vitest，38 项单测
npm run build      # 生产构建
```

资产生成脚本（读 .env 的 gpt-image 中转配置，均带重试/跳过已存在）：

```bash
node scripts/gen-seed-images.mjs     # 9 张种子梦境画（油画风）→ public/seeds-gen/
node scripts/gen-scene-textures.mjs  # 首页天幕 + 走廊尽头迷雾 → public/textures/
node scripts/gen-env-textures.mjs    # （已弃用产物 wall/floor，方向被证伪）
```

架构细节（分层、三层兜底、双边缘函数、降级系统）见根目录 [CLAUDE.md](../CLAUDE.md)。

## 3. 设计方向（三标杆）

| 标杆 | 借鉴什么 | 落地 |
|---|---|---|
| **noomo storytelling** | 视觉/画面/设计工艺：内容即视觉、构图、排版即画面 | 偏心构图、编辑式展签、大字排版、氛围图接管背景 |
| **kimi careers** | 交互编排：一格滚轮=一次完整幕间电影转场 | 首页分幕状态机、逐画注目节奏、入画俯冲 |
| **《光与影：33号远征队》** | 美术画风：Belle Époque 暗黑幻想油画（该游戏故事亦发生在画中，与本项目「画即世界」同构） | 种子图重生成为油画质感、鎏金装裱、明暗对照布光 |

核心理念：**画即世界** —— gpt-image 生成的内容不是页面里的一个元素，而是每一幕的主角；页面是画的内部。

## 7. 其他必要信息

- **gpt-image 中转**：base/model/key 在 gitignored `.env`（`OPENAI_BASE_URL` / `OPENAI_IMAGE_MODEL=gpt-image-2` / `OPENAI_API_KEY`）；单张 ~40s，连续调用易 524（脚本已带重试节流）。**Key 曾在对话中出现，建议轮换。**
- **质量门**：`check` + `lint`（零新增）+ `test`（38 项）+ `build` 全绿才提交；重要交互改动加 Playwright 无头实测
- **无头验证陷阱**：hash-only 的 `page.goto('#/xxx')` 不会重载文档，HMR 编辑中间态会污染已打开的页面——测试前必须先 `about:blank` 再导航，做文档级硬刷新；测本地存储迁移类逻辑时注意 `localStorage`/IndexedDB 状态需要精确模拟真实用户历史状态（onboarded、auto-loaded 标记等），否则测试路径与真实路径不一致会误判
- **验证环境注意**：dev 服务器由 AI 会话启动时，会话结束进程即死——页面连着死服务器会出现"样式未热更新/黑屏"类假象，遇异常先硬刷新或重启 `npm run dev`
